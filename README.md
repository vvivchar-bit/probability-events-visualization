# Практичне заняття №1  
## Основні поняття теорії ймовірностей

**Виконав:** Вівчар Вадим Вікторович  
**Група:** АЛК-43  
**Факультет:** Інженерія програмного забезпечення  

---

## Опис проєкту

Цей репозиторій містить візуалізацію операцій над подіями у теорії ймовірностей.

Практичний приклад побудовано на задачі, де технічний контроль перевіряє чотири вироби. Кожен виріб може бути якісним або бракованим.

---

## Позначення

- `Я` — якісний виріб;
- `Б` — бракований виріб;
- `Ω` — простір елементарних подій;
- `A` — хоча б один виріб є бракованим;
- `B` — серед перевірених виробів бракованих не менше двох;
- `Ā` — протилежна подія до A;
- `B̄` — протилежна подія до B;
- `A + B` — об'єднання подій;
- `AB` — перетин подій.

---

## Що реалізовано

У проєкті реалізовано:

- побудову простору елементарних подій для 4 виробів;
- візуальне виділення події `A`;
- візуальне виділення події `B`;
- знаходження протилежних подій `Ā` та `B̄`;
- знаходження об'єднання `A + B`;
- знаходження перетину `AB`;
- коротке пояснення кожної операції.
